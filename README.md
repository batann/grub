Custom Grub Bootloader Themes
======================================

🚀 Installation
---------------

Simply enter following commands in your terminal :

```bash
$ git clone https://github.com/batann/grub.git
$ cd grub
$ sudo ./install.sh
```

💻 Dependencies
---------------

**Custom Grub Themes** requires the following dependencies:

- git
